# Project Summary

This project was completed as part of a Machine Learning internship program.

The objective was to study telecom customer data and understand the factors responsible for customer churn. Exploratory Data Analysis (EDA) was performed to identify trends and relationships between customer attributes.

A Random Forest model was developed for churn prediction, while K-Means clustering was applied to segment customers into different groups. The findings can help businesses identify high-risk customers and improve retention strategies.

The project demonstrates practical applications of machine learning in customer analytics and business decision-making.
